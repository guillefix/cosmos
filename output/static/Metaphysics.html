This deduction uses tools like <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Epistemology.html">Epistemology</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Logic.html">Logic</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Science.html">Science</a>, etc. to infer our Knowledge of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Reality.html">Reality</a>, or Physical World.</li></ol><p>The physical world is based purely on primary substances (concrete things of the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Physics.html">physical world</a>.
