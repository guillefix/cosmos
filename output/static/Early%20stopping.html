<p>A <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Regularization">Regularization</a> technique where we have a validation data set, which we use for <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Cross-validation">Cross-validation</a>.
