<hr><p>In <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Category%20theory">Category theory</a>, this refers to</p><p><strong>Duality principle</strong>: every property/theorem has a corresponding <strong>dual</strong> property/theorem, by applying the theorem in the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Opposite%20category">Opposite category</a> and interpreting it in the original category. In other words, a statement S is true about C if and only if its dual (i.e. the one obtained from S by reversing all the arrows) is true about C op</p>
