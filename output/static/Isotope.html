<p>Isotopes are variants of a particular <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Chemical%20element">Chemical element</a> which differ in <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Neutron">Neutron</a> number, although all isotopes of a given element have the same number of <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Proton">Proton</a>s in each <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Atom">Atom</a>.</p>
