<p>Some features that are important in the behaviour of an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Evolution">evolving</a> system.</p><p><em>Important features</em>:</p><ul><li><strong>Evolvability</strong> (<a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Evolvability" rel="noopener noreferrer" target="_blank">wiki page</a>). May be defined as &quot;the capacity of a system for adaptive evolution&quot;; however, this is often called <strong>adaptability</strong> (I think). Evolvability is then defined just as &quot;the number of other phenotypes <em>potentially</em> accessible from a phenotype&quot; (see <a class="tc-tiddlylink-external" href="http://rsif.royalsocietypublishing.org/content/11/95/20140249" rel="noopener noreferrer" target="_blank">this paper</a> or <a class="tc-tiddlylink-external" href="http://rspb.royalsocietypublishing.org/content/275/1630/91" rel="noopener noreferrer" target="_blank">Robustness and evolvability: a paradox resolved</a>)</li><li><strong>Modularity</strong>. See <a class="tc-tiddlylink-external" href="http://journals.aps.org/pre/abstract/10.1103/PhysRevE.82.026117" rel="noopener noreferrer" target="_blank">Self-assembly, modularity, and physical complexity</a>, and <strong>complexity</strong>. <a class="tc-tiddlylink-external" href="http://az.oxfordjournals.org/content/36/1/36.abstract" rel="noopener noreferrer" target="_blank">Homologues, Natural Kinds and the Evolution of Modularity</a></li><li><strong>Robustness</strong>. See <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Mutational%20robustness">Mutational robustness</a>. <a class="tc-tiddlylink-external" href="http://www.youtube.com/watch?v=vLYpjUANUdc&amp;feature=youtu.be#t=4m40s" rel="noopener noreferrer" target="_blank">Environmental robustness</a>. There are genotypic and phenotypic robustness.
