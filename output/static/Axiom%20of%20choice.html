Axiom of choice

cosmos 25th September 2017 at 12:58pm

Given a collection A\mathcal{A}A of Disjoint nonempty Sets, there exists a set CCC consisting of exactly one element from each element of A\mathcal{A}A; that is, a set CCC such that CCC is contained in the union of the elements of A\mathcal{A}A, and for each A∈AA \in \mathcal{A}A∈A, the set C∩AC \cap AC∩A contains a single element.

See page 57 of Munkres 2nd ed.
