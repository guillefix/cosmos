<p>An <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Isomorphism">Isomorphism</a> between <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Formal%20system">Formal system</a>s (typographical systems), and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Number%20theory">Number theory</a>. </p><blockquote><p>CENTRAL PROPOSITION: If there is a typographical rule which tells how certain digits are to be shifted, changed, dropped, or inserted in any number represented decimally, then this rule can be represented equally well by an arithmetical counterpart which involves arithmetical operations with powers of 10 as well as additions, subtractions, and so forth.</p></blockquote><p>More briefly:</p><blockquote><p>Typographical rules for manipulating numerals are actually arithmetical rules for operating on numbers. </p></blockquote><p>This is central to the proof of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Godel%20incompleteness%20theorems">Godel incompleteness theorems</a>.</p><p>See also <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Godel%2C%20Escher%2C%20Bach">Godel, Escher, Bach</a></p><p><u><em>Producible numbers</em></u></p><p>Just as any set of typographical rules generates a set of theorems, a corresponding set of natural numbers will be generated by repeated applications of arithmetical rules. These producible numbers play the same role inside number theory as theorems do inside any formal system.</p>
