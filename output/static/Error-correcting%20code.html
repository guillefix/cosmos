number of letters in a code word misrepresented by noisy channel that can still be safely corrected at the receiving end. Thus, if the channel is not very noisy, a small d ( C ) (minimal <a class="tc-tiddlylink tc-tiddlylink-missing" href="Hamming%2520distance.html">Hamming distance</a> between two codewords) might suffice, otherwise larger values of d ( C ) are needed. When d ( C ) is large, this slows down the information transmission, because the source is not allowed to use all q n words for encoding, but still must transmit a sequence n -letter words</p><h2 class=""><u>Forward error correction</u></h2><p><a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Forward_error_correction" rel="noopener noreferrer" target="_blank">Forward error correction</a>: forward error correction (FEC) or channel coding[1] is a technique used for controlling errors in data transmission over unreliable or noisy communication channels, where the information flows only one way (see <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=ydzIhYbG3XI&amp;index=4&amp;list=PLE125425EC837021F" rel="noopener noreferrer" target="_blank">here</a>).. The central idea is the sender encodes the message in a redundant way by using an error-correcting code (ECC). The American mathematician Richard Hamming pioneered this field in the 1940s and invented the first error-correcting code in 1950: the Hamming (7,4) code.[2]</p><p>Two way error correction. Things like ARQ (automatic repeat request).</p><p><u>Main types of FEC codes</u>:</p><ul><li>Hamming codes</li><li>BCH codes (Reed-Solomon, convolutional coding)</li><li>Turbo codes</li><li>Galleger codes (LDPC)</li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Convolutional%2520code.html">Convolutional code</a> <a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Convolutional_code" rel="noopener noreferrer" target="_blank">Convolutional code</a> is a type of error-correcting code that generates parity symbols via the sliding application of a boolean polynomial function to a data stream. <a class="tc-tiddlylink-external" href="http://link.springer.com/article/10.1007%2Fs11432-015-5311-0" rel="noopener noreferrer" target="_blank">Stability of nonlinear feedback shift registers</a></li><li><a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Block_code" rel="noopener noreferrer" target="_blank">Block code</a> encode data in blocks.</li></ul><p>See <a class="tc-tiddlylink-external" href="http://pfister.ee.duke.edu/thesis/chap1.pdf" rel="noopener noreferrer" target="_blank">http://pfister.ee.duke.edu/thesis/chap1.pdf</a>, and other chapters.</p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=XjSue2cJWOU&amp;index=3&amp;list=PLE125425EC837021F" rel="noopener noreferrer" target="_blank">(IC 1.3) Applications of Error-correcting codes</a></p><h2 class=""><u>Quantum error correction</u></h2><hr><p><a class="tc-tiddlylink-external" href="https://link.springer.com/content/pdf/10.1007%2Fs00029-016-0284-4.pdf" rel="noopener noreferrer" target="_blank">Error correcting codes and neural networks</a></p></div> </div> </p> </section> </body> </html>
