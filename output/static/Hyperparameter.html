<p>In <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Learning%20theory">Learning theory</a>, a <strong>hyperparameter</strong> refers to a parameter in the learning algorithm itself, which isn't learned by the algorithm, but is chosen before running it. Hyperparameters may be learned (or <em>hyperlearned</em>), via methods of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Model%20selection">Model selection</a>.</p>
