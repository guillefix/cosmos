<p>A <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Differentiable%20function">differentiable mapping</a> between two <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Manifold">Manifold</a>s such that its <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Differential">Differential</a> is injective (i.e. the linear operator doesn't &quot;collapse&quot; any direction, and so its invertible)</p><p>If an immersion is a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Homeomorphism">Homeomorphism</a>, then it is an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Embedding">Embedding</a></p><p>For local properties it is enough to work with an immersion, because an immersion is locally (in a certain sense) an embedding. (Proposition 3.7 in <a class="tc-tiddlylink tc-tiddlylink-missing" href="#DoCarmo">DoCarmo</a>: given an immersion there is always a neighbourhood around each point such that the restriction to that nhbd is an embedding.
