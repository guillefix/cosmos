<p>A <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Representation">Representation</a> of a document used in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Text%20classification">Text classification</a>, which is a vector of fixed length. Each element corresponds to a word, and it is 1 if the word is present in the document, or 0 if it isn't.</p><p>There are variations where the number in the element represents the number of times the word appears in the document.</p>
