href="Dual.html">Dual</a>ity</u></h2><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Opposite%2520category.html">Opposite category</a>.</p><p><strong>Duality principle</strong>: every property/theorem has a corresponding dual property/theorem, by applying the theorem in the opposite category and interpreting it in the original category. In other words, a statement S is true about C if and only if its dual (i.e. the one obtained from S by reversing all the arrows) is true about C op</p><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Monic%2520morphism.html">Monic morphism</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Epic%2520morphism.html">Epic morphism</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Isomorphism.html">Isomorphism</a></p><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Initial%2520object.html">Initial object</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Terminal%2520object.html">Terminal object</a></p><p><strong>unique isomorphism property</strong>: There is a unique isomorphism between any pair of initial objects; thus initial objects are ‘unique up to (unique) isomorphism’, and we can (and do) speak of the initial object (if any such exists). By <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Dual.html">Dual</a>ity, any terminal object also satisfies this, because isommorphism is self-dual.</p><p><strong>Universal
