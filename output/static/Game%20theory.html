We map the outcomes to payoffs, which are numerical values where higher is better, and lower is worse.</p><p>A game <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=qQ3kFydI_xQ&amp;list=PL6EF60E1027E1A10B#t=7m30s" rel="noopener noreferrer" target="_blank">is composed of</a>:</p><ul><li>players</li><li>set of possible strategies for each player.
