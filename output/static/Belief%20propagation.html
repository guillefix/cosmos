<div class="tc-tiddler-body tc-reveal"><p><strong>Belief propagation</strong> can describe a particular kind of <a class="tc-tiddlylink tc-tiddlylink-missing" href="Message%2520passing.html">Message passing</a> process in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Network.html">Network</a>s, and the related algorithm for performing <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Inference.html">Inference</a> in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Conditional%2520random%2520field.html">Conditional random field</a>s. It is approximate (except for tree graphical models)</p><ul><li>The forward-backward algorithm of CRFs is a special case, it is belief propagation for the linear chain case. Belief propagation yields exact inference if the CRF graph is a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Tree%2520(combinatorial%2520structure).html">tree</a>, by computing marginals. <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=-z5lKPHcumo&amp;index=27&amp;list=PL6Xpj9I5qXYEcOhn7TqghAJ6NAPrNmUBH#t=22m40s" rel="noopener noreferrer" target="_blank">How to use</a></li><li>On graphs with loops, belief propagation can be used to do approximate inference. But one has to be careful with divergences (c.f. infinities in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Quantum%2520field%2520theory.html">Quantum field theory</a>, and loops in <a class="tc-tiddlylink tc-tiddlylink-missing" href="Feynman%2520diagram.html">Feynman diagram</a>s).</li></ul><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=-z5lKPHcumo&amp;index=27&amp;list=PL6Xpj9I5qXYEcOhn7TqghAJ6NAPrNmUBH" rel="noopener noreferrer" target="_blank">Neural networks [3.10] : Conditional random fields - belief propagation</a>. See <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Conditional%2520random%2520field.html">Conditional random field</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Factor%2520graph.html">Factor graph</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=-z5lKPHcumo&amp;index=27&amp;list=PL6Xpj9I5qXYEcOhn7TqghAJ6NAPrNmUBH#t=7m" rel="noopener noreferrer" target="_blank">Example</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=ASsKAaHlhCU" rel="noopener noreferrer" target="_blank">9 - 1 - Belief Propagation- Probabilistic Graphical Models - Professor Daphne Koller</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=nfB4Ucccws0&amp;list=PL04QVxpjcnjhe-E7LfEZ3SOvXSodbPNgu&amp;index=7#t=58m15s" rel="noopener noreferrer" target="_blank">video</a></p><h3 class=""><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=ASsKAaHlhCU" rel="noopener noreferrer" target="_blank">Belief propagation algorithm </a></h3><p>defines <strong>cluster graph</strong></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=nfB4Ucccws0&amp;t=3884s&amp;list=PL04QVxpjcnjhe-E7LfEZ3SOvXSodbPNgu&amp;index=7#t=59m11s" rel="noopener noreferrer" target="_blank">Belief propagation in statistical physics of constraint satisfaction problems</a>. In the context of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Statistical%2520physics.html">Statistical physics</a>, it's called <strong>cavity method</strong></p><p>In <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Coding%2520theory.html">Coding theory</a>, they call it <strong>sum-product algorithm</strong></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=8S6spV57P8U&amp;t=2199s&amp;list=PL04QVxpjcnjhe-E7LfEZ3SOvXSodbPNgu&amp;index=10" rel="noopener noreferrer" target="_blank">Belief propagation in factor graphs with factors involving two variables only</a></p><p>Cluster graphs must satisfy the <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=xwD_B31sElc" rel="noopener noreferrer" target="_blank">family preservation and the running intersection properties</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=xwD_B31sElc#t=11m" rel="noopener noreferrer" target="_blank">Bethe cluster graph</a> is a commonly used type of cluster graph, which comes from <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Statistical%2520physics.html">Statistical physics</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=gaiZ0N_gPoY" rel="noopener noreferrer" target="_blank">Convergence of belief propagation implies callibration (consistency among factors of marginal probabilities)</a></p><hr><p>Many general purpose libraries.</p><hr><p>See also related concepts/models in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Dynamical%2520systems%2520on%2520networks.html">Dynamical systems on networks</a></p></div> </div> </p> </section> </body> </html>
