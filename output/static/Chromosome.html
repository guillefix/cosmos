<p>A <strong>chromosome</strong> is a single molecule of DNA, containing many genes; an organisms often has several chromosomes. In a chromosome, the DNA is wound on <em>histone</em> proteins, and very densely packed, so that it can fit inside the nucleus. The packing structure is illustrated <a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Chromosome#/media/File:Chromatin_Structures.png" rel="noopener noreferrer" target="_blank">here</a>.</p>
