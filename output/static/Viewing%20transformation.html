<p>Part of the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Camera%20transform">Camera transform</a></p><p>Convert things in the viewing volume of the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Camera%20(computer%20graphics)">camera</a> (which is a <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Truncated%20pyramid">Truncated pyramid</a>) to <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Image%20space">Image space</a> (which is a <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Cube">Cube</a>)</p><p>This transformation is done so that the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Stereographic%20projection">Stereographic projection</a> that would have been necessary to project into the <em>image plane</em> in the original space, now becomes a <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Perpendicular%20projection">Perpendicular projection</a> in the image space!</p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=mpTl003EXCY&amp;index=5&amp;list=PL_w_qWAQZtAZhtzPI5pkAtcUVgmzdAP8g#t=16m" rel="noopener noreferrer" target="_blank">This can be done with a 4x4 matrix</a> – <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=mpTl003EXCY&amp;index=5&amp;list=PL_w_qWAQZtAZhtzPI5pkAtcUVgmzdAP8g#t=21m" rel="noopener noreferrer" target="_blank">this is the beginning of the matrix</a> – <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=mpTl003EXCY&amp;index=5&amp;list=PL_w_qWAQZtAZhtzPI5pkAtcUVgmzdAP8g#t=37m" rel="noopener noreferrer" target="_blank">this is the rest of the matrix</a></p><hr><p>In <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#OpenGL">OpenGL</a>, you can use <a class="tc-tiddlylink-external" href="https://www.khronos.org/registry/OpenGL-Refpages/gl2.1/xhtml/gluLookAt.xml" rel="noopener noreferrer" target="_blank">gluLookAt</a></p>
