<p>A collection of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Subset">Subset</a>s of a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Set">Set</a>, which cover (that is their <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Set%20union">union</a> is the whole set) the set and are <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Disjoint">Disjoint</a>.</p><p>The collection of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Equivalence%20class">Equivalence class</a>es of a set with an equivalence relation produces a <strong>partition</strong>. Furthermore, any partition induces a corresponding <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Equivalence%20relation">Equivalence relation</a></p>
