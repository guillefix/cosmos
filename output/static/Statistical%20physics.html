<p><strong>Statistical physics</strong> deals with the description of systems for which a deterministic description is either useless or impossible, so that one uses a <em>statistical</em> description.</p><p>Here a <em>deterministic description</em> is understood in the context of the relevant physical description. For example Schrodinger's equation is deterministic, if the relevant physical description is the wavefunction. It is non-deterministic if one takes position and/or velocity as the relevant physical descriptions. However, it is known that one can't describe quantum mechanical evolution purely with a statistical theory of position and velocity, without sacrificing some rather well-established physical principles or predictions.</p><p>If the system is effectively classical (either because it is macroscopic, or for some other reason, that is probably ultimately related to <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Quantum%20decoherence">Quantum decoherence</a>), the need for a statistical description arises when the system is <em>sufficiently chaotic</em>. Most often this requires the system to: have <u><big>many</big> components</u> and/or <u>be coupled to a system with <big>many</big> components</u>.</p><p>For this reason, statistical physics is mostly applied to the description of systems of many particles in a gas, liquid or solid; or to one or a few particles coupled to one such large system.</p><p>There are two main branches of statistical physics:</p><p><big><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Equilibrium%20statistical%20physics">Equilibrium statistical physics</a></big> deals with such systems at <em>equilibrium</em>, that is, when the relevant macroscopic averages of the statistical description don't change with time. In practice, one often has two approaches:</p><ul><li>For a small system coupled to a large chaotic system, one often has to use a probability distribution function over the relevant degrees of freedom (amazingly, for equilibrium, this always takes the form of a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Boltzmann%20distribution">Boltzmann distribution</a>.</li><li>For a large system, one can often bypass the distribution function and deal with the relevant averaged quantities directly, resulting in a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Thermodynamics">thermodynamic</a> description.</li></ul><p><big><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Non-equilibrium%20statistical%20physics">Non-equilibrium statistical physics</a></big> deals with such a system out of equilibrium, so that averages can change in time. This is much harder to do in full generality, as systems offer much more diversity out of equilibrium, as may be expected. One often has three approaches:</p><ul><li>For a small system coupled to a large chaotic system, one has a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Stochastic%20processes">stochastic process</a>, which describes the evolution under the random influence of the large chaotic system.</li><li>For a large system which is only slightly out of equilibrium, so that relevant macroscopic averages analogous to those used in thermodynamics can still be defined, one can describe the system using <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Non-equilibrium%20thermodynamics">Non-equilibrium thermodynamics</a></li><li>For a large system that is considerably out of equilibrium, one has to use the tools of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Kinetic%20theory">Kinetic theory</a> to describe it. However, if the system is <em>very</em> far from equilibrium, even these may be inappropriate, and finding an appropriate description may be extremely hard. An example of this are systems with strong <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Turbulence">Turbulence</a>. Our only approaches to understand these systems are often <a class="tc-tiddlylink tc-tiddlylink-missing" href="#phenomenological">phenomenological</a>.</li></ul><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=ov4KJwX2us0&amp;list=PL04QVxpjcnjhtL3IIVyFRMOgdhWtPn7YJ&amp;index=3" rel="noopener noreferrer" target="_blank">Intro vid to modern stat phys </a></p><hr><p>See also <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Complex%20systems">Complex systems</a>, and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Sloppy%20systems">Sloppy systems</a></p><p><a class="tc-tiddlylink-external" href="http://pages.physics.cornell.edu/~sethna/StatMech/EntropyOrderParametersComplexity.pdf" rel="noopener noreferrer" target="_blank">Entropy, Order Parameters, and Complexity</a></p><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Long-range%20interacting%20systems">Long-range interacting systems</a></p><p><a class="tc-tiddlylink-external" href="https://www-thphys.physics.ox.ac.uk/people/AlexanderSchekochihin/A1/" rel="noopener noreferrer" target="_blank">Oxford physics course</a></p><p><a class="tc-tiddlylink-external" href="https://www0.maths.ox.ac.uk/courses/course/28786/material" rel="noopener noreferrer" target="_blank">Oxford maths course</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/playlist?list=PL04QVxpjcnjjoHglJGm-O5YtSbqLtln0g" rel="noopener noreferrer" target="_blank">Bangalore School on Statistical Physics -
