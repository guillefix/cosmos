<p>You see, &quot;Mu&quot; is an ancient Zen answer which, when given to a question, UNASKs the question. Here, the question seems to be, &quot;Should the world be under stood via holism, or via reductionism?&quot; And the answer of &quot;MU&quot; rejects the premises of the question, which are that one or the other must be chosen. By unasking the question, it reveals <strong>a wider truth: that there is a larger context into which both holistic and reductionistic explanations fit</strong>. See the <strong>principle of inclusiveness</strong> in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Epistemology">Epistemology</a>.</p><p>Ideas of hyperreality, chaos magik, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Zen">Zen</a>, etc.</p><p><u>Etymology</u></p><p>I think it comes from a mathematical notation which uses <span><span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>μ</mi></mrow><annotation encoding="application/x-tex">\mu</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="strut" style="height:0.43056em;"></span><span class="strut bottom" style="height:0.625em;vertical-align:-0.19444em;"></span><span class="base textstyle uncramped"><span class="mord mathit">μ</span></span></span></span></span> to denote &quot;free&quot; searches (searches without bounds).
