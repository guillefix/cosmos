Well, I don't know if you know, but the Schrodinger equation can be written as a Fokker-Planck equation (up to an i). FP equation tends to Boltzmann distribution (maximum entropy distribution up to constraints). At the end of the day this would probably be equivalent to Feynman's way of deriving the principle of least action from path integrals (in there, one takes h–&gt;0. In the FP eq from the Schrodinger eq, h ~ 1/(diffusion constant), so h–&gt; 0 makes equilibration happen infinitely fast, so all the trajectory is at maximum entropy!) But I think the idea of using entropy maximization to do this, is pretty cool!
