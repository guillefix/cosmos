An Explosive percolation process that is based on chosing kkk vertices at random and adding edges among those vertices according to some rule.

kkk-vertex rules are actually a generalization of mmm-edge rules (aka Achlioptas process) because a mmm-edge rule can be constructed from a nnn-vertex rule, where n≥2mn \geq 2mn≥2m, which chooses n≥2mn\geq 2mn≥2m vertices at random (possibly repeating, but still being able to have mmm distinct edges), and then choose mmm edges at random within these 2m2m2m vertices. Note that we need n≥2mn\geq 2mn≥2m so that we don't restrict the chosen edges to have some vertex in common.

mmm-vertex rule (as defined here): In processes following an mmm-vertex rule, the agent is presented with the random list (set) vmv_mv​m​​ of vertices, and, unless two or more are already in the same component, must add one or more edges between them, according to any deterministic or random rule that depends only on the history.

Some kkk-vertex rules are examples of Non-self-averaging percolation process, showing novel supercritical phenomena, like stochastic staircases!

In Achlioptas process phase transitions are continuous, it was shown that the Percolation phase transition for processes following a vertex rule was continuous. However, they can still show some discontinuity arbitrarily close to the critical point (see Non-self-averaging percolation process).
