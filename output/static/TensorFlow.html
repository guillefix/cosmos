Note <code>None</code> can be used as a dimension, that can take any value (calculated from input data..)</li><li><strong>Model variables</strong> that are going to be optimized so as to make the model perform better. <code>weights = tf.Variable(initializationValues)</code></li><li><strong>The model</strong> which is essentially just a mathematical function that calculates some output given the input in the placeholder variables and the model variables. Example@ <code>logits = tf.matmul(x, weights) + biases</code></li><li>A <strong>cost function</strong> that can be used to guide the optimization of the variables. Example: <code>cross_entropy = tf.nn.softmax_cross_entropy_with_logits(logits=logits, labels=y_true)</code>, <code>cost = tf.reduce_mean(cross_entropy)</code></li><li>An <strong>optimization method</strong> which updates the variables of the model. Example: <code>optimizer = tf.train.GradientDescentOptimizer(learning_rate=0.5).minimize(cost)</code></li></ol></li><li>Begin a <strong>session</strong> (<code>tf.Session()</code>), and <code>train</code>, and <code>run</code> the computational graph, to get its result.<ol><li><code>session.run(tf.global_variables_initializer())</code></li><li><code>session.run(optimizer, feed_dict=feed_dict_train)</code>, where the keys in the feed_dict should match the placeholder variables.</li></ol></li></ol><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=bvHgESVuS6Q" rel="noopener noreferrer" target="_blank">Learning TensorFlow</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=2FmcHiLCwTU" rel="noopener noreferrer" target="_blank">TensorFlow in 5 Minutes</a></p><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=l4X-kZjl1gs" rel="noopener noreferrer" target="_blank">An Introduction to LSTMs in Tensorflow</a></p><hr><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=Lx8JUJROkh0&amp;index=4&amp;list=PL9Hr9sNUjfsmEu1ZniY0XpHSzl5uihcXZ&amp;spfreload=1#t=3m10s" rel="noopener noreferrer" target="_blank">How to save and restore model variables</a> – <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=Lx8JUJROkh0&amp;index=4&amp;list=PL9Hr9sNUjfsmEu1ZniY0XpHSzl5uihcXZ&amp;spfreload=1#t=5m55s" rel="noopener noreferrer" target="_blank">and here</a> – <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=Lx8JUJROkh0&amp;index=4&amp;list=PL9Hr9sNUjfsmEu1ZniY0XpHSzl5uihcXZ&amp;spfreload=1#t=11m30s" rel="noopener noreferrer" target="_blank">and restore</a></p>
