<p><em>aka &quot;self-replicating programs&quot;, &quot;self-reproducing programs&quot;, and &quot;self-copying programs&quot;</em></p><p>A quine is a non-empty computer program which takes no input and produces, when executed, a copy of its own source code as its only output.</p><p><a class="tc-tiddlylink-external" href="https://www.wikiwand.com/en/Quine_(computing)" rel="noopener noreferrer" target="_blank">wiki</a></p>
