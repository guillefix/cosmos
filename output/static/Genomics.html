<p>The study of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Genome">Genome</a>s of organisms</p><p><a class="tc-tiddlylink-external" href="https://www.wikiwand.com/en/Genomics" rel="noopener noreferrer" target="_blank">https://www.wikiwand.com/en/Genomics</a></p><p><a class="tc-tiddlylink-external" href="https://genome-euro.ucsc.edu/cgi-bin/hgGateway?hgsid=218799371_GrzgaLCTgLS4ixN0o64W7m83y2na" rel="noopener noreferrer" target="_blank">https://genome-euro.ucsc.edu/cgi-bin/hgGateway?hgsid=218799371_GrzgaLCTgLS4ixN0o64W7m83y2na</a></p><h2 class=""><u>Comparative genomics</u></h2><p>Has uses in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Biochemical%20genetics">Biochemical genetics</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Population%20genetics">Population genetics</a>, etc..</p><hr><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Bioinformatics">Bioinformatics</a></p><p><a class="tc-tiddlylink tc-tiddlylink-missing" href="#Haplotype%20inference">Haplotype inference</a> – Infering the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Haplotype">Haplotype</a>s (the two homologous chromosomes) from genomic data.</p><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#DNA%20sequencing">DNA sequencing</a> just gives all the base pairs at each position, but doesn't give which belong together in a single chromosome/haplotype.</p><p>Heterozygous <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Single-nucleotide%20polymorphism">SNP</a>s are the ones which are non-trivial to assign to the chromosome/haplotype.</p><ul><li>Clark's algorithm: a parsimony approach. Try to minimize the different number of haplotypes,</li><li>Statistical model approach. Find all potential haplotypes consistent with data. Model assigns a population frequency to each haplotype. If we know these frequencies we can assign the most probable haplotypes to each individual. <em>Missing data problem</em> –&gt; <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#EM%20algorithm">EM algorithm</a></li><li>Bayesian methods. Allow haploids which are different, but similar to influence each other's likelihood. <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Hidden%20Markov%20model">Hidden Markov model</a>.</li></ul>
