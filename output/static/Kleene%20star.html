<p>A <a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Kleene_star" rel="noopener noreferrer" target="_blank">Kleene star</a>, in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Mathematical%20logic">Mathematical logic</a> and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Computer%20science">Computer science</a>, (or Kleene operator or Kleene closure) is a unary operation, either on sets of strings or on sets of symbols or characters.</p><p>If V is a set of symbols or characters then V* is the set of all finite strings over symbols in V, including the empty string Îµ.</p><p>It is often used in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Coding%20theory">Coding theory</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Formal%20language">Formal language</a> theory, etc.</p>
