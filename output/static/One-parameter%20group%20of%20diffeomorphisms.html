<p>One can find a correspondence between <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Vector%20field">Vector field</a>s (under some assumptions) and a <strong>one-parameter group of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Diffeomorphism">Diffeomorphism</a>s</strong>, so that the vector field corresponds to the &quot;infinitesimal displacement&quot; of the flow represented by this group, the parameter being time.</p><p>See <a class="tc-tiddlylink-external" href="https://courses.maths.ox.ac.uk/node/view_material/33094#page=28" rel="noopener noreferrer" target="_blank">here</a> for the formal definition and proof of correspondence. The correspondence <a class="tc-tiddlylink-external" href="https://courses.maths.ox.ac.uk/node/view_material/33094#page=30" rel="noopener noreferrer" target="_blank">uses</a> integral curves to build the One-parameter group of diffeomorphisms given a vector field.</p><p>A <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Curve">Curve</a> in the manifold whose <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Tangent%20vector">Tangent vector</a> at every point is the vector of the vector field at that point is called an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Integral%20curve">Integral curve</a> </p>
