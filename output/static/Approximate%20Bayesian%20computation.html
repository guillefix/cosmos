<p>When even the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Likelihood%20function">Likelihood function</a> is intractable to compute.., but the model can be (stochastically) simulated. Similar regime as for <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Probabilistic%20programming">Probabilistic programming</a>?</p><p>In ABC, we simulate the system with different values of the parameters (sampled from prior), and compute some summary <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Statistic">Statistic</a>s. If the summary statistic is close enough to that of the observed data, we accept that run, and save the parameter values that generated it.</p><p>We end up with a sample of parameter values which is our approximate posterior.</p><p>There are more advanced variants: sequential particle MC... </p>
