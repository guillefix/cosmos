<p><img src="http://i.imgur.com/i0vZ63R.png"></p><p>A filter subbase can generate a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Filter%20base">Filter base</a>, and like it, it can be extended so that a family of filter subbases (which we call <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Subbasis">subbase</a>) to generate a whole <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Topological%20space">Topological space</a>.</p><p>Note that the sets forming the subbase are part of the base they generate, because finite intersections include the intersection of a set with itself.</p>
