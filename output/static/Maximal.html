<p>A maximal element of an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Partially%20ordered%20set">Partially ordered set</a> os am element that is not smaller than any other element in the set. A minimal element of is defined dually as an element that is not greater than any other element in the set.</p><p>Often, we talk about maximal with respect to a set, in which case we often implicitly talk about the partial ordering given by the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Subset">Subset</a> relation. </p><p><a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Maximal_element" rel="noopener noreferrer" target="_blank">https://en.wikipedia.org/wiki/Maximal_element</a></p>
