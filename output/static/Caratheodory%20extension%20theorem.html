<p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=WKdy_OxdUbk#t=15m23s" rel="noopener noreferrer" target="_blank">Video</a></p><p>To specify a measure on a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Sigma-algebra">Sigma-algebra</a> it suffices to specify it on an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Algebra%20(algebraic%20structure)">Algebra (algebraic structure)</a>.</p><p>The measure then extends to the sigma algebra generated by that algebra, i.e. to the smallest sigma-algebra containing that algebra. The generation can be done by starting with the algebra and taking intersections and unions, so that it satisfies the axioms of a sigma-algebra. The generated sigma-algebra is unique, if the underlying measure is <a class="tc-tiddlylink-external" href="https://www.wikiwand.com/en/%CE%A3-finite_measure" rel="noopener noreferrer" target="_blank">sigma-finite</a>..</p>
