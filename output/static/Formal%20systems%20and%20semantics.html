Semantics</a></p><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Semantics.html">Semantics</a> – the study of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Meaning.html">Meaning</a>. Hofstadter argues that meaning is a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Morphism.html">Morphism</a> (often an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Isomorphism.html">Isomorphism</a>) between a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Formal%2520language.html">language</a> (produced by a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Formal%2520system.html">Formal system</a>) and another space. The mapping itself is known as an interpretation.</p><p><strong>Meaningfull</strong> interpretations are those where there is an identifications between the theorems and non-theorems of the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Formal%2520system.html">Formal system</a>, and a pair of sets that correspond to some aspects of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Reality.html">Reality</a>, for instance a set of true and false statements. See page 51 of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Godel%252C%2520Escher%252C%2520Bach.html">GEB</a>. An interpretation is otherwise known as <em>meaningless</em>.</p><p><em>symbols of a formal system, though initally without meaning, cannot avoid taking on &quot;meaning&quot; of sorts, at least if an isomorphism is found.</em> However, unlike <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Natural%2520language.html">Natural language</a> and other non-<a class="tc-tiddlylink tc-tiddlylink-resolves" href="Formal%2520system.html">Formal system</a>s, we can't add new rules or axioms because the meaning we have found; by definition of <em>formality</em>, the rules are fixed. To distinguish these we call these active and passive <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Meaning.html">Meaning</a>s</p></div> </div> </p> </section> </body> </html>
