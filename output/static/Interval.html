<p>A set bounded by two elements in an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Ordering">ordered</a> set. If it includes the bounds, it's a <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Closed%20interval">Closed interval</a>, otherwise, it's an <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Open%20interval">Open interval</a>.</p>
