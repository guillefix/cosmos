<p>An <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Enzyme">Enzyme</a> that helps <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Glyceraldehyde%203-phosphate">Glyceraldehyde 3-phosphate</a> become <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Dihydroxy%20acetone%20phosphate">Dihydroxy acetone phosphate</a>.</p><p>See <a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=SrkK-wJ7j_I" rel="noopener noreferrer" target="_blank">video from edx course</a>.</p><p>It is a diffusion-limited enzyme (the rate-limiting step in the reaction is <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Diffusion">Diffusion</a> itself). This is also called kinetically perfect. There is no point in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Evolution">Evolution</a> to make it any faster, as it's not rate-limiting.</p>
