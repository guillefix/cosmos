<p>Classical decision-time algorithm in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Artificial%20intelligence">Artificial intelligence</a>. For each state encountered, a large tree of possible continuations is considered. The approximate value function is applied to the leaf nodes and then <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Value%20function%20backup">backed up</a> toward the current state at the root.</p><p>See sec 8.9 in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Sutton-Barto">Sutton-Barto</a></p>
