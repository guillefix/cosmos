</div> </div> <div class="tc-tiddler-body tc-reveal"><p><strong>Soft condensed matter</strong> (often abbreviated to <strong>soft matter</strong>) is basically all forms of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Condensed%2520matter%2520physics.html">condensed matter</a> (i.e. many particles more or less bound together (for e.g., by <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Intermolecular%2520forces.html">Intermolecular forces</a>)) that isn't a solid, so that it has features that are easily deformable at low energies (room thermal energies). This includes <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Polymer%2520physics.html">polymers</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Liquid%2520crystals.html">Liquid crystals</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Complex%2520fluid%2520dynamics.html">complex fluids</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Granular%2520material.html">Granular material</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Foam.html">Foam</a>s, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Emulsion.html">Emulsion</a>s, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Colloid.html">Colloid</a>s, and many kinds of mixtures, that form mesoscopic structures. Also a lot of stuff in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Biology.html">life</a> falls under the &quot;soft&quot; category. I like it precisely because of its richness.</p><p>Wiki: <a class="tc-tiddlylink-external" href="https://en.wikipedia.org/wiki/Soft_matter" rel="noopener noreferrer" target="_blank">https://en.wikipedia.org/wiki/Soft_matter</a></p><p><u>Typial features</u></p><ul><li>Mesoscopic lengthscales</li><li>Fluctuations (<a class="tc-tiddlylink tc-tiddlylink-resolves" href="Brownian%2520motion.html">Brownian motion</a>)</li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Self-assembly.html">Self-assembly</a>. <a class="tc-tiddlylink tc-tiddlylink-missing" href="Block-copolymer%2520self-assembly.html">Block-copolymer self-assembly</a></li></ul><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Statistical%2520physics.html">Statistical physics</a> is important, in particular interplay of energy and entropy, reflected in the free energy.</p><p>For systems of many particles, one uses <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Statistical%2520field%2520theory.html">Statistical field theory</a>. Though one can further simplify by ignoring fluctuations, using a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Mean%2520field%2520theory.html">Mean field theory</a>.</p><p><u><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Soft%2520materials.html">Soft materials</a></u></p><ul><li>Simple liquids (fully homogeneous/isotropic). See <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Fluid%2520mechanics.html">Fluid mechanics</a>. The fluid may be a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Solution%2520(Chemistry).html">solution</a>, but that only changes it <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Chemistry.html">chemical</a> properties, not its physical ones.</li><li>Several types of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Complex%2520fluid%2520dynamics.html">complex fluids</a>. Main types:<ul><li>Solid–liquid (<a class="tc-tiddlylink tc-tiddlylink-resolves" href="Suspension.html">Suspension</a>s (particularly <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Colloid.html">Colloid</a>s) and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Gels.html">Gels</a>)</li><li>solid–gas (<a class="tc-tiddlylink tc-tiddlylink-resolves" href="Granular%2520material.html">Granular material</a>s, and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Fibrous%2520material.html">Fibrous material</a>s)</li><li>liquid–gas (<a class="tc-tiddlylink tc-tiddlylink-resolves" href="Foam.html">Foam</a>s)</li><li>liquid–liquid (<a class="tc-tiddlylink tc-tiddlylink-resolves" href="Emulsion.html">Emulsion</a>s)</li></ul></li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Liquid%2520crystals.html">Liquid crystals</a>: nematic, smectic, cholesteric, columnar, etc.</li><li>Several <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Polymer.html">polymeric</a> materials are soft. E.g. some <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Plastic.html">Plastic</a>s, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Rubber.html">Rubber</a>s, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Latex.html">Latex</a></li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Surfactant.html">Surfactant</a>s</li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Biological%2520matter.html">Biological matter</a></li><li>Many kinds of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Active%2520matter.html">Active matter</a> and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Driven%2520matter.html">Driven matter</a>.</li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="Pastes.html">Pastes</a>, or <a class="tc-tiddlylink-external" href="http://journals.aps.org/prl/abstract/10.1103/PhysRevLett.78.2020" rel="noopener noreferrer" target="_blank">soft glassy materials</a></li></ul><p>A fruitful way of studying phases, is to study the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Phase%2520transition.html">phase transitions</a> between them.</p><p>Universality, coarse-graining, renormalization group. <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Percolation.html">Percolation</a></p><hr><p><a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=kvDxOhEVvpQ&amp;index=26&amp;list=PL04QVxpjcnjjDk2ESwBZ1sEcKu91wDV5l" rel="noopener noreferrer" target="_blank">Introduction to soft matter physics - 1 by David Pine</a></p><p><a class="tc-tiddlylink-external" href="https://scholar.google.co.uk/citations?hl=en&amp;user=u2KmUD0AAAAJ&amp;view_op=list_works&amp;sortby=pubdate" rel="noopener noreferrer" target="_blank">Thomas Speck</a></p></div> </div> </p> </section> </body> </html>
