<p>A <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Path%20(Graph%20theory)">path</a> through the edges of a simplicial complex, or a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Delta-complex">Delta-complex</a>, where the path returns to the same vertex.</p><p>When drawing the path, if we draw a part of a path, and then we draw the same part but backwards, then they &quot;cancel out&quot;, meaning, that the path is the same as removing those parts..</p><p>There are higher dimensional analogues.</p>
