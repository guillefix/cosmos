<p>Second-order logic is an extension of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#First-order%20logic">First-order logic</a> where, in addition to variables which can range over objects (elements of the domain class), there are variables which can take <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Predicate">Predicate</a>s as values. This means we can <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Quantifier">quantify</a> over predicates too. <em>Predicate</em>s are also called <em>Properties</em>, and in set-theoretic <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Semantics">Semantics</a> (see <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Model%20theory">Model theory</a>), it means subsets of the domain class.</p><p><a class="tc-tiddlylink-external" href="https://math.stackexchange.com/questions/2483821/differentiating-first-second-order-logic" rel="noopener noreferrer" target="_blank">https://math.stackexchange.com/questions/2483821/differentiating-first-second-order-logic</a></p><p><a class="tc-tiddlylink-external" href="https://plato.stanford.edu/entries/logic-higher-order/" rel="noopener noreferrer" target="_blank">https://plato.stanford.edu/entries/logic-higher-order/</a></p><p><a class="tc-tiddlylink-external" href="https://math.stackexchange.com/questions/108415/what-is-the-relationship-between-zfc-and-first-order-logic" rel="noopener noreferrer" target="_blank">https://math.stackexchange.com/questions/108415/what-is-the-relationship-between-zfc-and-first-order-logic</a></p><p><a class="tc-tiddlylink-external" href="https://www.lesswrong.com/posts/SWn4rqdycu83ikfBa/second-order-logic-the-controversy" rel="noopener noreferrer" target="_blank">https://www.lesswrong.com/posts/SWn4rqdycu83ikfBa/second-order-logic-the-controversy</a></p>
