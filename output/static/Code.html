Can prove using a tree construction (see <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Kraft%20inequality">Kraft inequality</a>), <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Probabilistic%20method">Probabilistic method</a>, or others..</p><p>If the source alphabet itself is made of strings of a fixed length from another alphabet, these are called <em>blocks</em>, and the code is called a <strong>block code</strong>.</p><p>Otherwise we have a <strong>symbol code</strong>.</p>
