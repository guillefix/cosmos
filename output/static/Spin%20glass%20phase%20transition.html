<p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Phase%20transition">Phase transition</a>s in <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Spin%20glass">Spin glass</a>es</p><p>the greatest theoretical success in spin glasses to date has to do with the analysis and solution of a spin glass in an (effectively) infinite number of dimensions, where we have the only theoretical results that there is a phase transition</p><p>It is easy to prove theoretically that an EA spin glass in one dimension has no phase transition (but this is true for any one-dimensional system with short-range interactions only)</p><p>From numerical simulations, the current consensus is that, for the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Edwards-Anderson%20Hamiltonian">EA Ising model</a>, there is a phase transition (at nonzero temperature) in four dimensions and higher, probably one in three dimensions as well, and none in two.</p><p>See <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Edwards-Anderson%20Hamiltonian">Edwards-Anderson Hamiltonian</a> for a description of the order parameter, and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Replica%20symmetry%20breaking">Replica symmetry breaking</a>.</p>
