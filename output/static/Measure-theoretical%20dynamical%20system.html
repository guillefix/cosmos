<p>A <strong>Measure-theoretical dynamical system</strong> is comprised of:</p><ul><li>a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Measurable%20space">Measurable space</a></li><li>a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Measurable%20function">measurable map</a></li><li>a non-singular <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Measure">Measure</a></li></ul><p>This space can be considered, without restriction to be a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Probability%20space">Probability space</a>. See Amigo's book.</p><p>See <a class="tc-tiddlylink-external" href="https://www.dropbox.com/s/fv3fc6jawf9oswl/%5BManfred_Denker%5D_Ergodic_Theory_on_Compact_Spaces%28BookZZ.org%29.djvu?dl=0" rel="noopener noreferrer" target="_blank">here</a> (<a class="tc-tiddlylink-external" href="file:///home/guillefix/Dropbox/COSMOS/Mathematics/Ergodic%20theory/%5BManfred_Denker%5D_Ergodic_Theory_on_Compact_Spaces(BookZZ.org).djvu" rel="noopener noreferrer" target="_blank">local</a>).</p><p>A <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Dynamical%20system">Dynamical system</a> on a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Measurable%20space">Measurable space</a> has a <em>natural</em> or <em>physical invariant measure</em>, corresponding to the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Probability%20measure">Probability measure</a> that numerical simulations of the system would produce asymptotically.</p>
