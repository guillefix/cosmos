<p>A representation is something that is intimately &quot;<a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Relation">related</a>&quot; to something else, so that one can say that it <em>represents</em> it.</p><p>In <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Computer%20science">Computer science</a>, representation is crucial, because <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Information">Information</a> needs to represent something else to have <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Meaning">Meaning</a>, by definition. This meaning, is what makes <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Mind">Mind</a>s and computers &quot;think&quot; and be useful and not just process meaningless information (which they also do).</p><p>Art: <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Design">Design</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Medium">Medium</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Communication">Communication</a></p><p>Mathematical formalisms: <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Coding%20theory">Coding theory</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Information%20theory">Information theory</a></p><p>Philosophy: <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Language%20and%20meaning">Language and meaning</a>, <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Philosophy%20of%20mind">Philosophy of mind</a>.</p>
