<p>The <strong>genotype</strong> is the set of <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Allele">Allele</a>s corresponding to a particular <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Gene">Gene</a> of an individual (<a class="tc-tiddlylink-external" href="https://www.youtube.com/watch?v=H6gwYdyn7II#t=1m40s" rel="noopener noreferrer" target="_blank">vid</a>). Sometimes, it can refer to the set of alleles of a set of genes, I think. The set of alleles for all genes, is called the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Genome">Genome</a>.</p><p><u><a class="tc-tiddlylink-external" href="https://global.britannica.com/science/ploidy" rel="noopener noreferrer" target="_blank">Ploidy</a></u></p><ul><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Haploid">Haploid</a>, if the genotype contains a single allele.</li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Diploid">Diploid</a>, if the genotype contains a two alleles.</li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Polyploid">Polyploid</a>, if the genotype contains a more than two alleles.</li></ul><p><u>Zygosity</u></p><ul><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Homozygote">Homozygote</a>, if the genotype is diploid and the two alleles are the same.</li><li><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Heterozygote">Heterozygote</a>, if the genotype is diploid and the two alleles are the same.</li></ul>
