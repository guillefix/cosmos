<p>A <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Perturbation">Perturbation</a> to a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Structure">Structure</a> that <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Motion">moves</a>.</p><p>In <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Mechanics">Mechanics</a> and <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Condensed%20matter%20physics">Condensed matter physics</a>, it refers to <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Displacement">Displacement</a> of the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Atom">Atom</a>s relative to the equilibrium positions in the <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Crystal%20lattice">Crystal lattice</a> in a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Crystal">crystalline material</a>. More generally, it refers to variations in the particle density relative to the mean density.</p><p>When the dispersion relation of the pressure waves is linear, the waves are called <a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Sound">Sound</a>.</p><p><a class="tc-tiddlylink tc-tiddlylink-resolves" href="#Mathematics">Mathematically</a>, it is often described via a <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Wave%20equation">Wave equation</a>. This can be derived as the continuum limit of a system of <a class="tc-tiddlylink tc-tiddlylink-missing" href="#Coupled%20oscillators">Coupled oscillators</a>.</p>
