An <strong>abstract simplicial complex</strong> <span><span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>K</mi></mrow><annotation encoding="application/x-tex">K</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="strut" style="height:0.68333em;"></span><span class="strut bottom" style="height:0.68333em;vertical-align:0em;"></span><span class="base textstyle uncramped"><span class="mord mathit" style="margin-right:0.07153em;">K</span></span></span></span></span> is just a set of points and subsets of these points satisfying some simple conditions. Its <strong>topological realization</strong>, denoted <span><span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi mathvariant="normal">∣</mi><mi>K</mi><mi mathvariant="normal">∣</mi></mrow><annotation encoding="application/x-tex">|K|</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="strut" style="height:0.75em;"></span><span class="strut bottom" style="height:1em;vertical-align:-0.25em;"></span><span class="base textstyle uncramped"><span class="mord mathrm">∣</span><span class="mord mathit" style="margin-right:0.07153em;">K</span><span class="mord mathrm">∣</span></span></span></span></span>, is a corresponding <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Topological%2520space.html">Topological space</a> gotten from the disjoint union of simplices associated with these subsets, quotient certain inclusion relations (similar but not the same as <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Delta-complex.html">Delta-complex</a>es).</p><p>We also define a <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Triangulation.html">Triangulation</a> of a space, as well as <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Simplicial%2520map.html">Simplicial map</a>s between simplicial complexes.</p><p>A more flexible way of constructing topological spaces are <a class="tc-tiddlylink tc-tiddlylink-resolves" href="CW%2520complex.html">CW complex</a>es.</p><p>Can be used to calculate <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Fundamental%2520group.html">Fundamental group</a>s</p><hr><p>Traditionally, simplicial homology is defined for simplicial complexes, which are the <a class="tc-tiddlylink tc-tiddlylink-resolves" href="Delta-complex.html">∆ complexes</a> whose simplices are uniquely determined by their vertices. This amounts to saying that each n simplex has n + 1 distinct vertices, and that no other n simplex has this same set of vertices.</p></div> </div> </p> </section> </body> </html>
